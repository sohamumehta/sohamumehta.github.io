A Visual Tool for the Simplex Method in LP

CS 284 (Fall 2012) (Instructor: Prof. Satish Rao)

This is a User Interface to visualize (1) a convex polyhedron defined by a set of linear constraints in 3 variables,
and (2) the simplex method to maximize an objective functions subject to the these constraints. Details
on the method can be obtained here. The visualization was enabled with OpenGL, and the UI was designed using GLUI.
