Assignment 1: Deformable Body Simulation

I used the method of "Position Based Dynamics" first proposed by Matthias Muller in this paper. The basic structures are (1) a node or particle which is the smallest element of the deformable body which moves according to external forces and constraints; and (2) the constraints which represent conditions that must be met by nodes 'joined' by the constraint. For the current implementation, the only constraint used was a spring-like constraint between joined nodes

(1) Cloth Simulation: The cloth is a 2D grid of nodes interconnected with adjacent, next to adjacent and diagonal constraints, under gravity.

For resolving self-collisions, the constraint-based method is extremely complex and slow (due to multiple iterations of constraint projection). So, I instead used a simpler method as described in this Paper, to implement self-collision correction.


(2) Jello simulation: The jello object is a 3D solid rectangular grid of nodes interconnected with adjacent, next to adjacent, diagonal and body-diagonal constraintsconstraints, under gravity.
